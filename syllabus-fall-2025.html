2 Course Description

Welcome to ISM4641: Python for Business Analytics!
This course provides a comprehensive introduction to using Python for data analysis and predictive modeling, specifically tailored for business applications. You will learn essential programming skills, data manipulation and visualization techniques, and foundational statistical analysis methods. In the latter part of the course, you will also be introduced to machine learning concepts, including both linear and logistic regression, enabling you to apply these tools to real-world business challenges.

Formal Course Description: (Insert official course description from the USF catalog here).

Prerequisites:
No prior programming experience is required; however, a willingness to learn and a basic understanding of statistical concepts are beneficial.

Course Modality:
This is a 16‑week hybrid course with a unique structure: - Mondays: In-person lectures and demonstrations (required attendance) - Thursdays: Remote lab/practice sessions (optional live help via MS Teams, or work independently)

The course includes a midterm exam (Week 6) and a final exam (Week 16). Each week comprises Monday lectures and Thursday practice exercises/labs.

3 Course Objectives and Learning Outcomes

By the end of this course, students will be able to:

  • Programming Fundamentals:
    • Write Python code for data cleaning, manipulation, and analysis.
    • Utilize libraries such as NumPy and Pandas for efficient data handling.
    • Apply fundamental programming constructs (variables, data types, operators, control flow, and functions).
  • Data Visualization:
    • Create effective visualizations using Matplotlib and Seaborn.
    • Customize plots to communicate analytical insights clearly.
  • Statistical Analysis:
    • Calculate and interpret descriptive statistics and data distributions.
    • Perform hypothesis testing and construct confidence intervals.
  • Predictive Modeling:
    • Build, evaluate, and interpret linear and logistic regression models using Scikit-learn.
    • Understand model evaluation metrics and the bias-variance tradeoff.
  • Business Applications:
    • Apply Python-based techniques to solve real-world business problems.
    • Effectively communicate the results of your analyses.
