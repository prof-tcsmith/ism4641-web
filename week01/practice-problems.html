Section 1: Variables and Data Types

Concept Review

  • Variables are like containers that store data. They have names and hold values.
  • Data Types specify the kind of value a variable holds (e.g., text, numbers, True/False).
  • Common data types:
    • str (string): For text, enclosed in single (’ ’) or double (” “) quotes.
    • int (integer): For whole numbers.
    • float (float): For numbers with decimal points.
    • bool (Boolean): For True or False values.

Section 2: Operators

Concept Review

  • Operators perform operations on variables and values.
  • Arithmetic Operators: +, -, *, /, // (floor division), % (modulus), ** (exponentiation).
  • Comparison Operators: == (equal to), != (not equal to), > (greater than), < (less than), >= (greater than or equal to), <= (less than or equal to).
  • Logical Operators: and, or, not.
