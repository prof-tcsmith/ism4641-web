Solution 2:

def calculate_discounted_price(price, discount_percentage):
    """Returns the final price after applying the discount."""
    discount_amount = price * (discount_percentage / 100)
    final_price = price - discount_amount
    return final_price

# Test the function
result = calculate_discounted_price(200, 15)
print(result)  # Expected output: 170.0

Solution 6:

def business_report(title, *args, report_date="Unknown", **kwargs):
    """Generates a formatted business report."""
    print(f"Report Title: {title}")
    print(f"Report Date: {report_date}")
    print("\nKey Performance Indicators:")
    for kpi in args:
        print(f" - {kpi}")
    print("\nAdditional Metrics:")
    for metric, value in kwargs.items():
        print(f" {metric}: {value}")

# Test the function
business_report(
    "Monthly Sales Report",
    "Total Sales",
    "New Customers",
    report_date="2025-03-01",
    Revenue="$500K",
    Profit="$120K",
    Growth="15%"
)

Solution 7:

Local variables are defined within a function and exist only during the function’s execution, while global variables are defined outside of any function and can be accessed throughout the program.

Example:

global_var = "I am global"

def example():
    local_var = "I am local"
    print(global_var)  # Accessible inside the function
    print(local_var)   # Accessible inside the function

example()
# print(local_var)  # This would raise an error because local_var is not accessible outside the function.

Create a file named financial_tools.py with the following content:

# financial_tools.py

def calculate_roi(initial_investment, final_value):
    """
    Calculate the ROI.
    
    ROI = ((Final Value - Initial Investment) / Initial Investment) * 100
    """
    return ((final_value - initial_investment) / initial_investment) * 100

def calculate_roi(initial_investment, final_value):
    """Calculates ROI using the formula: ((final_value - initial_investment) / initial_investment) * 100."""

Solution 14:

def detailed_greeting(name, *args, **kwargs):
    """Prints a detailed greeting message incorporating additional greetings and attributes."""
    print(f"Hello, {name}!")
    if args:
        print("Additional greetings:")
        for msg in args:
            print(f" - {msg}")
    if kwargs:
        print("Attributes:")
        for key, value in kwargs.items():
            print(f" {key}: {value}")

# Test the function
detailed_greeting("Alice", "Welcome aboard", "Good to see you", title="Dr.", time_of_day="morning")

Solution 15:

Docstrings are string literals placed as the first statement in a function. They serve to document what the function does, its parameters, and its return value. They are important because they help other developers (and your future self) understand how to use the function without needing to read its implementation.
