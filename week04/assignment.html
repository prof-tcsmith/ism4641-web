Assignment 4: Advanced Business Analytics Functions and Modules

Worth: 10 Points

Business Case: TechTime Gadgets Promotional Analysis

The Story:
TechTime Gadgets is an innovative online retailer specializing in the latest electronic gadgets. In an effort to boost sales and refine its promotional strategies, the management has decided to analyze both current pricing structures and sales performance. They need a flexible, modular Python program that not only calculates discounted prices and aggregates sales data but also generates a timestamped report and computes the Return on Investment (ROI) from recent promotions.

The Problem:
As a member of the business analytics team, you are tasked with developing a Python solution that incorporates the advanced function concepts (including default arguments, *args, and **kwargs) and module usage from Week 4. Your program should:

  1. Calculate Discounted Prices:
    Create a function that calculates the final price of a product after applying a discount percentage. This function should have default arguments where appropriate.

  2. Aggregate Sales Data and Generate a Report:
    Develop a function that accepts a variable number of sales figures using *args and additional promotional metrics (like “Total Orders”, “New Customers”, etc.) via **kwargs. This function should:

    • Sum the provided sales figures.
    • Incorporate the additional metrics into a formatted report.
    • Include a timestamp in the report using Python’s datetime module.

  3. Compute ROI from Promotional Data:
    Create a separate custom module named sales_tools.py that contains a function calculate_roi(initial_investment, final_value) which computes the ROI using the formula:

    \[ \text{ROI} = \left(\frac{\text{Final Value} - \text{Initial Investment}}{\text{Initial Investment}}\right) \times 100 \]

    Import and use this module in your main script to compute the ROI based on provided promotional data.

  4. Main Function Integration:
    Write a main() function that ties all of these elements together. This function should:

    • Call your discounted price function for a sample product.
    • Use your report generation function to display the aggregated sales and promotional metrics.
    • Call the ROI function from your custom module to compute and display the ROI.
    • Print the complete, timestamped report to the console.

Provided Data

For the purposes of this assignment, use the following sample data:

  • Product Pricing and Discount Data:

    • Price: $799.99
    • Discount Percentage: 10%

  • Sales Figures:
    Sales over the past week (in dollars): 1500.0, 2300.5, 1890.75, 2120.0

  • Promotional Metrics (as keyword arguments):

    • Total_Orders: 125
    • New_Customers: 30
    • Conversion_Rate: “5%”

  • Promotional Investment Data for ROI Calculation:

    • Initial Investment: $5000
    • Final Value (Revenue from Promotion): $8500

Your Tasks

  1. Discount Calculation Function (2 Points):
    • Write a function calculate_discounted_price(price, discount_percentage=10) that returns the final price after discount.
    • Test the function using the provided price and discount data.
  2. **Sales Report Function with *args and kwargs (3 Points):
    • Write a function generate_sales_report(*sales, report_date="Unknown", **promo_metrics) that:
      • Sums all sales figures passed via *args.
      • Integrates the keyword arguments into a report.
      • Retrieves the current date and time (formatted as "YYYY-MM-DD HH:MM:SS") and uses it as a timestamp in the report.
    • Test the function with the provided sales figures and promotional metrics.
  3. Custom Module for ROI (2 Points):
    • Create a module file named sales_tools.py containing a function calculate_roi(initial_investment, final_value) that returns the ROI percentage.
    • In your main script, import this module and compute the ROI using the provided investment data.
  4. Main Function Integration (3 Points):
    • Write a main() function that:
      • Calls calculate_discounted_price and prints the discounted price.
      • Calls generate_sales_report and prints the generated report.
      • Uses sales_tools.calculate_roi to compute and print the ROI.
    • Ensure that running your script executes the main() function and displays the complete report.
