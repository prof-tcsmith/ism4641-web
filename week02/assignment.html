Introduction

This assignment tests your understanding of control flow (conditional statements and loops) and data structures (lists and dictionaries) in Python, which were covered in Week 2 of ISM4641. You will be creating a simplified e-commerce system that manages a product catalog, customer orders, and applies discounts based on certain criteria.

Scenario:

You are tasked with creating a small part of an e-commerce system for an online store. The system needs to:

  1. Store product information (product ID, name, price).
  2. Store customer information (customer ID, name, loyalty status).
  3. Allow customers to create orders by selecting products and quantities.
  4. Calculate the total order value.
  5. Apply discounts based on the customer’s loyalty status and the order total.
  6. Print a summary of the order.

System Requirements

1. Data Storage

  • Product Catalog: Use a dictionary to store product information.
    • Key: Product ID (string)
    • Value: A dictionary containing the product’s name (string) and price (float).
  • Customer Database: Use a dictionary to store customer information.
    • Key: Customer ID (string)
    • Value: A dictionary containing the customer’s name (string) and loyalty status (Boolean, True if a loyalty member, False otherwise).

2. Order Creation

  • Prompt the user to enter their customer ID. (HINT: see the input function from week02 notes)
  • Check if the customer exists in the customer database. If not, print an error message and exit.
  • Create an empty list called order_items to store the items in the order.
  • Use a while loop to allow the user to add items to the order:
    • Display the product catalog to the user (product IDs, names, and prices).
    • Prompt the user to enter a product ID.
    • Prompt the user to enter the quantity.
    • If the product ID is valid:
      • Add a dictionary to the order_items list. This dictionary should contain the product ID, name, price, and quantity.
    • If the product ID is invalid, print an error message.
    • Ask the user if they want to add another item to the order (yes/no). If the answer is “no”, exit the loop.

3. Discount Calculation

  • Use an if-elif-else structure to determine the discount based on the following rules:
    • Loyalty Member & Order Total >= $100: 15% discount
    • Loyalty Member & Order Total < $100: 10% discount
    • Non-Loyalty Member & Order Total >= $100: 5% discount
    • Non-Loyalty Member & Order Total < $100: No discount

4. Order Summary

  • Use a for loop to iterate through the order_items list and calculate the total order value before the discount.
  • Calculate the final order total after applying the discount.
  • Print an order summary that includes:
    • Customer ID and name
    • A list of the items ordered (product ID, name, quantity, and price per item)
    • The total order value before the discount
    • The discount amount
    • The final order total
