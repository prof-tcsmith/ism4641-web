Control Flow and Data Structures: Practice Problems

This problem set is designed to reinforce your understanding of control flow (conditional statements and loops) and data structures (lists and dictionaries) in Python, as covered in Week 2 of ISM4641.

Instructions:

  • Try to solve each problem on your own first.
  • Write your code in a Python environment (e.g., Google Colab, Jupyter Notebook, or a local Python environment).
  • Test your code thoroughly with different inputs to ensure it works correctly.
  • Refer to the lecture notes and the Python documentation if you need help.
  • Example solutions are provided in the Appendix at the end of this document. However, try your best to solve the problems independently before looking at the solutions.

Problems

Conditional Statements (if, elif, else)

1. Discount Eligibility:

A store offers a 10% discount if a customer’s purchase totals $100 or more. Write a program that takes the purchase total as input and prints the final price after applying the discount (if applicable).

2. Grade Conversion:

Write a program that takes a numerical grade (0-100) as input and prints the corresponding letter grade based on the following scale: * 90-100: A * 80-89: B * 70-79: C * 60-69: D * Below 60: F

3. Shipping Cost:

An online store calculates shipping costs based on the following rules: * Orders $75 or more: Free shipping * Orders between $50 and $74.99: $5 shipping * Orders between $25 and $49.99: $10 shipping * Orders below $25: $15 shipping

Write a program that takes the order total as input and prints the shipping cost.

4. Employee Bonus:

A company determines employee bonuses based on their performance rating and years of service. * “Excellent” rating and 5+ years: 15% bonus * “Excellent” rating and 3-4 years: 10% bonus * “Good” rating and 3+ years: 5% bonus * Otherwise: No bonus

Write a program that takes the performance rating and years of service as input and prints the bonus percentage.

5. Leap Year:

Write a program that takes a year as input and determines if it’s a leap year. A leap year is divisible by 4, but not divisible by 100 unless it’s also divisible by 400.

Loops (for, while)

6. Sales Total:

You have a list of daily sales figures: sales = [120, 150, 130, 180, 200, 160, 190]. Use a for loop to calculate the total sales for the week.

7. Average Calculation:

Write a program that takes a list of numbers as input and calculates the average using a for loop.

8. Number Guessing Game:

Write a program that generates a random number between 1 and 100. Then, use a while loop to repeatedly ask the user to guess the number. Provide feedback (higher/lower) after each guess. End the loop when the user guesses correctly.

9. Factorial:

Write a program that takes an integer as input and calculates its factorial using a while loop. (Factorial of n is the product of all positive integers up to n).

10. List Reversal:

Write a program that takes a list as input and reverses it using a for loop and the insert() method. Do not use the reverse() method or slicing for this problem.

Lists

11. Positive Numbers:

Write a program that takes a list of numbers as input and creates a new list containing only the positive numbers from the original list.

12. Even Numbers:

Write a program that takes a list of numbers as input and creates a new list containing only the even numbers from the original list.

13. List Duplicates:

Write a program that takes a list as input and removes duplicate elements, creating a new list with unique elements only.

14. Common Elements:

Write a program that takes two lists as input and creates a new list containing only the elements that are common to both lists.

15. Second Largest:

Write a program that takes a list of numbers as input and finds the second largest number in the list.

Dictionaries

16. Product Inventory:

Create a dictionary to represent a product inventory. Each key should be a product name (string), and the value should be the quantity in stock (integer). Write a program that allows the user to: * Add new products to the inventory. * Update the quantity of existing products. * Print the current inventory.

17. Customer Orders:

Create a dictionary to store customer orders. Each key should be a customer name (string), and the value should be a list of order IDs (strings). Write a program that allows the user to: * Add new customers. * Add orders to a customer’s order list. * Print all orders for a given customer.

18. Word Frequency:

Write a program that takes a string of text as input and creates a dictionary where the keys are the unique words in the text, and the values are the number of times each word appears.

19. Student Grades:

Create a dictionary to store student grades. Each key should be a student name (string), and the value should be another dictionary with subject names as keys and grades as values. Write a program that: * Calculates the average grade for each student. * Prints the student with the highest average grade.

20. Contact List:

Create a dictionary to represent a contact list. Each key should be a contact name, and the value should be another dictionary with keys for “phone” and “email”. Write a program that allows the user to: * Add new contacts. * Search for a contact by name and print their phone and email. * Delete a contact.

        break
    else:
        print("Invalid action.")

# 19. Student Grades
student_grades = {}

while True:
    action = input("Enter 'add', 'calculate', 'print', or 'quit': ")
    if action == 'add':
        student_name = input("Enter student name: ")
        student_grades[student_name] = {}

        while True:
            subject = input("Enter subject name (or 'done'): ")
            if subject == 'done':
                break
            grade = float(input(f"Enter grade for {subject}: "))
            student_grades[student_name][subject] = grade
    elif action == 'calculate':
      for student_name, grades in student_grades.items():
        average_grade = sum(grades.values()) / len(grades) if grades else 0
        print(f"{student_name}: Average grade = {average_grade:.2f}")
    elif action == 'print':
      highest_average_grade = -1
      top_student = ""
      for student_name, grades in student_grades.items():
          average_grade = sum(grades.values()) / len(grades) if grades else 0
          if average_grade > highest_average_grade:
              highest_average_grade = average_grade
              top_student = student_name
      print(f"Student with highest average grade: {top_student} ({highest_average_grade:.2f})")

    elif action == "quit":
        break
