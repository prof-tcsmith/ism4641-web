Binary Classification Model-Selection Scenarios

Confusion-Matrix Reasoning Practice

Author

ISM4641

Published

April 21, 2025

Below are ten stand-alone business scenarios.
For each scenario you receive the confusion matrices of Model A and Model B (validation data).

Student task
1. Read the scenario introduction & note what “positive” means.
2. Decide which error type (FP or FN) is costlier and choose the metric that should dominate (Precision, Recall, F1, Accuracy).
3. Pick the model you would deploy and justify the choice.

Scenario 1 – Credit-Card Fraud Detection

Context. A bank’s real-time fraud engine flags suspicious transactions for manual review.

  • Positive class (1) = “Transaction is fraudulent.”
  • Negative class (0) = “Transaction is legitimate.”

Metric Formulas

Metric Formula Interpretation
Precision TP / (TP + FP) Of predicted positives, fraction correct.
Recall TP / (TP + FN) Of actual positives, fraction caught.
F1-score 2 × (P×R)/(P+R) Harmonic mean balancing Precision & Recall.
Accuracy (TP + TN) / Total Overall correctness (can mislead if classes are imbalanced).
